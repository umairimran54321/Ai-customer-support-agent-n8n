# Ai-customer-support-agent-n8n
AI-powered customer support agent built with n8n. Features RAG-based knowledge retrieval, conversation memory, human escalation, and website chat integration. Easily customizable for business support workflows.
Technologies Used

* n8n
* OpenAI API
* OpenAI Embeddings
* Simple Vector Store
* Recursive Character Text Splitter
* Default Data Loader
* Airtable (optional)
* HTTP Webhooks
* JSON
* REST APIs

Use Cases

This workflow can be used for:

* E-commerce customer support
* SaaS customer service
* Appointment booking assistants
* Internal company knowledge assistants
* FAQ automation
* Product support
* Educational institutions
* Healthcare information desks
* Real estate inquiries
* Small business customer service

Knowledge Base

The AI retrieves answers from business-specific documents instead of relying only on general AI knowledge. This significantly improves response accuracy and reduces hallucinations.

Supported knowledge sources include:

* FAQs
* Company policies
* Product documentation
* Shipping information
* Return policies
* PDF documents
* Text files
* Website content
* Internal documentation

Human Escalation

If the AI cannot confidently answer a customer’s question, the workflow automatically flags the conversation for human assistance. This prevents incorrect information from being provided while ensuring a smooth customer experience.
